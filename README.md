# Panda-Discord-Bot
Ez itt egy elég kicsi ám annál jobb magyar discorrd bot amiben van 1-2 jó dolog.
Példáúl:
-Economy (gazdaság)
-Hibarendszer
-Prémium mód
-Hitelesítő rendszer (PRÉMIUM)
-Üdvözlő üzenet
-Globális report rendszer
